# Ảnh Cơ sở vật chất

Thả 3 file ảnh vào thư mục này, đặt **đúng tên** sau (chạy ngay, không cần sửa code):

| Tên file | Vị trí hiển thị |
|---|---|
| `facility-1.jpg` | Ảnh lớn nổi bật (bên trái) |
| `facility-2.jpg` | Ảnh nhỏ (phải trên) |
| `facility-3.jpg` | Ảnh nhỏ (phải dưới) |

## Kích thước khuyến nghị
- `facility-1.jpg` (ảnh lớn): **1600 × 1000 px** · tỉ lệ ngang ~16:10
- `facility-2.jpg`, `facility-3.jpg` (ảnh nhỏ): **800 × 600 px** · tỉ lệ ngang ~4:3
- Định dạng JPG/WebP, nén dưới ~300KB/ảnh.
- Ảnh tự `object-cover` nên không méo dù tỉ lệ khác (chỉ cắt nhẹ mép).

## Sửa nhanh trong Admin
Vào **Admin → Website → Cơ sở vật chất** (`/cms/facilities`) để:
- Upload ảnh trực tiếp (hoặc dán URL)
- Sửa tiêu đề / mô tả / caption
- Ẩn/hiện cả section, thêm/xóa ảnh
- Xem gợi ý kích thước từng ảnh ngay trong giao diện
